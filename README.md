# godocument
Tài liệu Go tiếng Việt (Vietnamese Go Documentation)

## Giới thiệu

Dự án này cung cấp bản dịch tiếng Việt của tài liệu chính thức ngôn ngữ lập trình Go. Mục tiêu là giúp các lập trình viên Việt Nam tiếp cận và học Go dễ dàng hơn.

## Nội dung

- [doc.md](doc.md) - Trang tài liệu chính (dịch từ [go.dev/doc](https://go.dev/doc/))

### Các phần chính trong tài liệu:
- **Bắt đầu**: Cài đặt Go, các hướng dẫn cơ bản
- **Sử dụng và hiểu Go**: Effective Go, đặc tả ngôn ngữ, mô hình bộ nhớ
- **Tham chiếu**: Bản phát hành, cam kết tương thích Go 1, Wiki
- **Bảo mật**: Chính sách bảo mật, quản lý lỗ hổng, govulncheck
- **Công cụ chẩn đoán**: Garbage collector, profiling
- **Truy cập Cơ sở Dữ liệu**: Làm việc với database/sql
- **Cgo**: Liên kết mã Go với C
- **Các chủ đề nâng cao**: Coverage, fuzzing, assembly, race detector
- **Đo lường từ xa (Telemetry)**: Cấu hình và quyền riêng tư
- **Xây dựng và phát hành**: Module mirror, private modules
- **Phát triển Go**: Đóng góp mã, phát triển
- **Cộng đồng**: Diễn đàn, Slack, meetups

## Đóng góp

Chúng tôi hoan nghênh mọi đóng góp! Nếu bạn muốn cải thiện bản dịch hoặc thêm nội dung mới, vui lòng tạo Pull Request.

## Introduction (English)

This project provides Vietnamese translations of the official Go programming language documentation. The goal is to help Vietnamese developers access and learn Go more easily.

## Contents

- [doc.md](doc.md) - Main documentation page (translated from [go.dev/doc](https://go.dev/doc/))

### Main sections in the documentation:
- **Getting Started**: Installing Go, basic tutorials
- **Using and Understanding Go**: Effective Go, language specification, memory model
- **References**: Release history, Go 1 compatibility promise, Wiki
- **Security**: Security policy, vulnerability management, govulncheck
- **Diagnostics**: Garbage collector, profiling
- **Database Access**: Working with database/sql
- **Cgo**: Linking Go code with C
- **Advanced Topics**: Coverage, fuzzing, assembly, race detector
- **Telemetry**: Configuration and privacy
- **Building and Releasing**: Module mirror, private modules
- **Developing Go**: Contributing code, development
- **Community**: Forum, Slack, meetups
